  ### Installation

To install follow these steps.

1.  Clone the repo using either HTTPS

```
https://github.com/Codeyg12/reduxStore.git
```

or SSH key

    git@github.com:Codeyg12/reduxStore.git

2.  install NPM packages and seed

```
npm i && npm run seed
```

3. Run start

```
npm run start
```
